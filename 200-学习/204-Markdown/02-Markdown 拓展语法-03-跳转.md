# 跳转

## 标题之间跳转

```markdown
<!-- 同一文档跳转 -->
[跳转名称](#标题名称)

<!-- 不同文档跳转 -->
[跳转名称](文件夹路径#标题名称)
```

**注意**：

- 标题名称之间有空格，需要用 `-` 代替
- 标题名称之间有 `：`，不需要写
